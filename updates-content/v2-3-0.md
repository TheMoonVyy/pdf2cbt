### Enhanced Line Crop Mode of Test Maker

Added a new setting **Auto-skip Bottom Line On Next Page** to **Test Maker** which can be accessed by clicking on settings icon on the left-side panel.

This setting is made to address the issue in Line Crop Mode as mentioned in [this reddit post](https://www.reddit.com/r/PDF2CBT/comments/1s6riw4/can_we_do_something_to_resolve_this/).

This setting is only applicable when using Line Crop Mode.

- when **Auto-skip Bottom Line On Next Page** is turned on, it will automatically turn on **Skip Next Bottom Line** when moving to next page (this is indicated with the line turning red). This avoids having to manually skip the bottom line when moving to next page each time when the PDF has footers/unwanted stuff that requires to be skipped at the end of each page.
- When **Auto-skip Bottom Line On Next Page** is turned off, it won't automatic turn on **Skip Next Bottom Line**. Useful for PDFs that don't have footer/unwanted stuff at the end of each page (i.e. questions are truly continuous across pages).

**From v2.3.0** onwards, this setting is turned **on by default**.

In previous versions (v2.0.0 to v2.2.0), cropper behaved the same as when this setting is turned off.
